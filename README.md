# Amazon Web Scraper 🛒

A robust Python-based web scraping tool designed to extract product information from Amazon. This project demonstrates efficient data collection using **Requests** and structured data processing using **Pandas**.

---

## 🚀 Features
* **Automated Extraction:** Scrapes product titles, prices, ratings, and availability.
* **Data Cleaning:** Uses Pandas to handle missing values and format currency.
* **Export Ready:** Saves processed data into clean CSV or Excel formats.
* **Custom Headers:** Implements user-agent rotation to mimic browser behavior.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** * `Requests`: For handling HTTP requests and fetching HTML content.
    * `Pandas`: For data manipulation, cleaning, and structured storage.
    * `BeautifulSoup4`: For parsing HTML elements.
